HTML элемент определяется открывающим тегом, некоторым содержимым и закрывающим тегом.

HTML элементы

HTML элемент - это все от открывающего тега до закрывающего тега:

<tagname>Содержимое расположено здесь<tagname>

Примеры нескольких HTML элементов:

<h1>Мой первый заголовок<h1>

<p>Мой первый абзац.<p>


Открывающий тэг Содержимое элемента Закрывающий тэг
<h1> Мой первый заголовок </h1>
<p1> Мой первый абзац. </p1>
<br> none none

Примечание: Некоторые HTML элементы не имеют содержимого (как элемент <br>). Эти элементы называются пустыми. У пустых элементов нет закрывающего тэга


Вложенные HTML элементы

HTML элементы могут быть вложенными (это означает, что элементы могут включать в себя другие элементы)

Все HTML документы содержат в себе вложенные HTML элементы

Следующий пример содержит четыре HTML элемента ( <html>, <body>, <h1> and <p> ):

Пример

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>
