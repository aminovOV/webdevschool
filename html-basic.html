В этой главе мы покажем некоторые базовые примеры HTML.

Не волнуйтесь если мы используем теги, которые вы пока еще не выучили.

HTML документы

Все HTML документы должны начинаться с обявления типа документа: <!DOCTYPE html>.

Сам HTML документ должен начинаться с <html> и заканчиваться </html> .

Видимая часть HTML документа находится между тегами <body> и </body> .

Пример

<!DOCTYPE html>
<html>
<body>


<h1>My First Heading</h1>
<p>My first paragraph.</p>


</body>
</html>

Объявление <!DOCTYPE>

Объявление <!DOCTYPE> представляет тип документа и помогает браузеру правильно отображать веб-страницы.

Оно должно быть указано только один раз, наверху страницы (до первого тега HTML).

Объявление <!DOCTYPE> не чувствительно к регистру.

Объявление <!DOCTYPE> для HTML5 выглядит так:

<!DOCTYPE html>

Заголовки HTML

Заголовки HTML определяются тегами <h1>-<h6>.

<h1> определяет самый важный заголовок. <h6> определяет наименее важный заголовок:

Пример

<h1> Это заголовок 1<h1>

<h2> Это заголовок 2<h2>

<h3> Это заголовок 3<h3>

Абзацы HTML

Абзацы HTML определяются тегом <p>:

Пример

<p> Это абзац.<p>

<p> Это другой абзац.<p>

Ссылки HTML

Ссылки HTML определяются тегом <a>:

Пример

<a href="https://www.google.com"> Это ссылка.</a>


Попробуй сам »

Адрес ссылки указывается в атрибуте <href>.

Атрибуты используются для предоставления дополнительной информации о HTML элементах.

Вы узнаете больше об атрибутах в дальнейших главах.

Изображения HTML

Изображения HTML определяются тегом <img>.

Исходный файл (<src>), альтернативный текст (<alt>), width and height are provided as attributes:

Пример

<img src="webdevschool.jpg" alt="webdevschool.ru" width="104" height="142">

Как просмотреть исходный код HTML?

Вы когда-нибудь видели веб-страницу и удивлялись: "Эй! Как они это сделали?"

Просмотр исходного кода HTML:

Правый клик по странице HTML, выбрать "Просмотр кода страницы" (в браузере "Chrome") или "Просмотреть исходный код" (в Edge), или что-то похожее в других браузерах. Откроется окно, содержащее исходный код HTML страницы.

Просмотр исходного кода HTML элемента:

Правый клик по элементу (или пустой области), выбрать "Просмотреть код", чтобы увидеть из чего состоят элементы (вы увидите одновременно и HTML и CSS). Вы также можете редактировать HTML и CSS на лету в панели элементов или стилей.
