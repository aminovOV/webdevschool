HTML - стандартный язык гипертекстовой разметки для веб-страниц.

Что такое HTML?

HTML расшифровывается как гипертекстовый язык разметки

HTML - это стандартный язык разметки для создания веб-страниц

HTML описывает структуру веб-страницы

HTML состоит из ряда элементов

HTML элементы сообщают браузеру, как отображать содержимое

HTML элементы помечают фрагменты содержимого, такие как "это заголовок", "это абзац", "это ссылка" и т.д.


Простой HTML документ

Пример

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>


<h1>This is a Heading</h1>
<p>This is a paragraph.</p>


</body>
</html>

Объяснение примера

Объявление <!DOCTYPE html> определяет, что этот документ является документом HTML5

Элемент <html> - это корневой элемент HTML страницы

Элемент <head> содержит мета-информацию о HTML странице

Элемент <title> задает заголовок для HTML страницы (который показывается в строке заголовка браузера или на вкладке страницы

Элемент <body> определяет тело документа и является контейнером для всех видимых элементов, абзацев, изображений, гиперрсылок, таблиц, списков и т.д.

Элемент <h1> определяет большой заголовок

Элемент <p> определяет абзац

Что такое HTML элемент

HTML элемент определяется открывающим тэгом, некоторым содержимым и закрывающим тэгом

<tagname>Содержимое расположено здесь<tagname>

HTML элемент-это все от открывающего до закрывающего тэга

<h1>Мой первый заголовок<h1>

<p>Мой первый абзац.<p>


Открывающий тэг Содержимое элемента Закрывающий тэг
<h1> Мой первый заголовок </h1>
<p1> Мой первый абзац. </p1>
<br> none none

Некоторые HTML элементы не имеют содержимого (как элемент <p1>). Эти элементы называются пустыми. У пустых элементов нет закрывающего тэга



Веб-браузеры

Назначение веб-браузеров (Chrome, Edge, Firefox, Safari) - прочитать HTML документ и корректно отобразить его.

Браузер не отображает HTML тэги, но использует их, чтобы определить, как отобразить документ.
