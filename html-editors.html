Простой текстовый редактор - это все, что вам нужно для изучения HTML.

Изучайте HTML, используя Блокнот или TextEdit

Веб-страницы могут создаваться и изменяться с использованием профессиональных HTML редакторов.

Однако, для изучения HTML мы рекомендуем простой текстовый редактор, как Блокнот(для ПК) или TextEdit (для Mac).

Мы считаем, что использование простого текстового редактора - лучший способ изучать HTML.

Следуйте шагам, изложенным ниже, чтобы создать свою первую веб-страницу с Блокнот или TextEdit.

Шаг 1: Откройте Блокнот (для ПК)

Для Windows 8 или более поздних версий:

Откройте стартовый экран (символ окна, в нижнем углу вашего экрана). Введите "Блокнот".

Для Windows 7 или более ранних версий:

Откройте Старт > Программы > Стандартные > Блокнот

Шаг 1: Откройте TextEdit(для Mac)

Откройте Finder > Приложения > TextEdit

Далее поменяйте некоторые настройки, чтобы приложение сохраняло файлы правильно. В разделе Персональные настройки > Формат > выберите"Обычный текст".

Далее под пунктом "Открыть и сохранить", поставьте галочку напротив "Отображать HTML файлы, как HTML код вместо фоматированного текста".

Затем откройте новый документ, чтобы поместить код.

Шаг 2: Написать некоторый HTML

Напишите или скопируйте следующий HTML код в Блокноте:

<!DOCTYPE html>
<html>
<body>


<h1>My First Heading</h1>

<p>My first paragraph.</p>


</body>
</html>

Шаг 3: Сохраните HTML страницу

Сохраните файл на своем компьютере. Выберите Файл > Сохранить как в меню Блокнота.

Назовите файл "index.htm" и установите кодировку UTF-8 (которая является предпочтительной для HTML файлов).


Примечание: Вы также можете использовать .htm или .html в кач


Шаг 4: Просмотрите HTML страницу в браузере

Откройте сохраненный HTML файл в вашем любимом браузере (двойной клик на файле или правый клик и выбрать "Открыть с помощью").
